[attr] Sélectionne tous les éléments ayant déclaré un attribut « attr » <input type="text" required> [required] {
  border: 2px solid red;
}
[attr=val] Sélectionne tous les éléments ayant déclaré un attribut « attr » avec la valeur « val » <a href="https://example.com">Lien</a> [href="https://example.com"] {
  color: orange;
}
Lien vers example.com
[attr~=val] Sélectionne tous les éléments ayant déclaré un attribut « attr » contenant le mot « val » (mot entier) <p class="info important">Texte</p> [class~="important"] {
  font-weight: bold;
}

Texte avec classe "important"
[attr^=val] Sélectionne tous les éléments ayant déclaré un attribut « attr » commençant par la sous-chaine « val » <a href="https://site.com">Lien sécurisé</a> [href^="https"] {
  border-left: 3px solid green;
}
Lien sécurisé (HTTPS)
[attr$=val] Sélectionne tous les éléments ayant déclaré un attribut « attr » se terminant par la sous-chaine « val » <a href="document.pdf">PDF</a> [href$=".pdf"] {
  color: #e74c3c;
}
document.pdf
[attr*=val] Sélectionne tous les éléments ayant déclaré un attribut « attr » contenant la sous-chaine « val » <img src="photo.jpg" alt="Une belle photo"> [alt*="belle"] {
  border: 2px solid gold;
}
Image avec "belle" dans l'attribut alt

Les Pseudo-classes

Elles peuvent être insérées après un sélecteur. La pseudo-classe agira sur tous les éléments (compatibles) du document.

Pseudo-classe Description Exemple HTML Exemple CSS Résultat / Explication
:hover Cible un élément survolé par la souris <button>Survolez-moi</button> button:hover {
  background: #3498db;
}

(Passez la souris sur le bouton)
:nth-child(N) Cible le Nième élément enfant <ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>		 li:nth-child(2) {
  color: red;
}
  • Item 1
  • Item 2
  • Item 3
:not(selecteur) Cible les éléments qui ne correspondent pas au sélecteur entre parenthèses <p class="texte">Texte normal</p>
<p class="special">Texte spécial</p>		 p:not(.special) {
  opacity: 0.7;
}

Texte normal

Texte spécial (non affecté)
/* Autres pseudo-classes importantes */
a:link { color: blue; } /* liens non visités */
a:visited { color: purple; } /* liens visités */
input:focus { border-color: #3498db; } /* élément focus */
input:disabled { opacity: 0.5; } /* éléments désactivés */
input:checked + label { font-weight: bold; } /* cases cochées */

Les Pseudo-éléments

Ils peuvent être insérés après un sélecteur. Le pseudo-élément agira sur tous les éléments (compatibles) du document.

Pseudo-élément Description Exemple HTML Exemple CSS Résultat
::before Permet d'insérer un contenu avant un élément et de le styler « à la volée » <p class="note">Information importante</p> .note::before {
  content: "ℹ ";
  color: blue;
}

ℹ Information importante

::after Permet d'insérer un contenu après un élément et de le styler « à la volée » <span class="price">99</span> .price::after {
  content: " €";
  font-weight: bold;
}
99 €
::first-letter Permet de styler la première lettre d'un élément <p>Un paragraphe avec une première lettre stylisée.</p> p::first-letter {
  font-size: 2em;
  color: red;
}

Un paragraphe avec une première lettre stylisée.

Exemple Complet

Voici un exemple qui combine plusieurs sélecteurs :

<!-- HTML -->
<nav class="main-nav">
  <ul>
    <li><a href="#">Accueil</a></li>
    <li class="active"><a href="#">Services</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>

/* CSS */
.main-nav ul {
  list-style: none;
  padding: 0;
}

.main-nav li {
  display: inline-block;
  margin: 0 10px;
}

.main-nav a {
  text-decoration: none;
  padding: 5px 10px;
  border-radius: 3px;
}

.main-nav a:hover {
  background-color: #3498db;
  color: white;
}

.main-nav li.active a {
  background-color: #2c3e50;
  color: white;
}
